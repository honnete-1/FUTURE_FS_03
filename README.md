# Net Moviex – AI-Inspired Brand Reimagining

This project is a **creative rebranding of Netflix**, built as **Task 3** for the  
[Future Intern Full Stack Web Development Internship](https://futureinterns.com/full-stack-web-development-task-3/).

## Live Demo

[Visit Net Moviex](https://futurefs03-sandy.vercel.app/)

---

## Project Summary

As part of this challenge, I was tasked with **reimagining and rebuilding a famous brand's website** using modern web technologies and a creative AI-inspired twist.

I chose to rebrand **Netflix** into **Net Moviex** — a futuristic streaming platform lookalike with unique visuals, improved UX, and a dynamic feel.

---

##  Technologies Used

### Frontend
- **React** (with Vite)
- **Framer Motion** – animations and transitions
- **React Icons**
- **React Router DOM**

### Backend
- **Node.js** + **Express**
- **MySQL** – for database management
- **dotenv**, **CORS**

### Deployment
- **Frontend**: Vercel  
- **Backend**: Render  
- **Database**: MySQL (locally hosted or cloud)

---

##  Key Features

- AI-inspired UI design for a futuristic look  
- Dynamic movie card rendering  
- Scroll and hover animations using Framer Motion  
- Interactive navigation and hero sections  
- Modular and reusable React components  
- Full frontend–backend separation and communication

- How to creatively rebrand a tech product using **UI/UX principles**
